# Contributing to FwTA

Thanks for your interest in Fix With Ticket Agent!

## Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating, you agree to uphold it.

## Development

1. Clone the repo.
2. `node src/index.js init .` to set up.
3. Make changes.
4. Test with `node demo/server.js` and the agent flows (simulation via llm_response.json).
5. Open PR.

## Adding features

- Keep it secure and open-source safe (see SAFETY.md).
- Update AGENT.md and docs/ if LLM-visible behavior changes.
- Prefer minimal, auditable changes.
- Add tests or verification steps where possible.

## Questions?

Open an issue or see [AGENT.md](./AGENT.md) for LLM usage patterns.
